# AI智学助手——智能教学辅助系统

## 一、项目简介

AI智学助手是一款面向教师与学生的智能教学辅助系统，系统围绕课程管理、学生学习、教师发布学习任务、AI辅助生成题目、学生答题记录、学习进度查看和智能学习分析等功能展开。

本系统旨在构建“课前预习—课中检测—课后复习”的闭环式教学辅助平台。教师可以通过系统创建课程、发布学习任务、查看学生学习进度；学生可以完成课程学习、提交答题记录并查看个人学习情况。系统结合大模型能力，实现题目智能生成和教学内容辅助分析，提高教师备课效率和学生个性化学习效果。

## 六、运行方式

### 1. 启动后端项目

进入后端项目目录：

    learning-assistant-backend

使用 IntelliJ IDEA 打开后端项目，确认数据库配置正确后，运行 Spring Boot 启动类。

也可以在终端中执行：

    mvn spring-boot:run

后端默认运行地址一般为：

    http://localhost:8080

具体端口以 application.properties 或 application.yml 中配置为准。

### 2. 启动客户端项目

进入客户端项目目录：

    learning-assistant-client

使用 IntelliJ IDEA 打开 JavaFX 客户端项目，运行主启动类。

客户端启动后，需要保证后端服务已经正常运行，否则登录、注册、课程加载等功能无法正常访问。

## 七、配置说明

后端需要配置数据库连接信息，例如：

    spring.datasource.url=jdbc:mysql://localhost:3306/数据库名
    spring.datasource.username=你的数据库用户名
    spring.datasource.password=你的数据库密码

如果使用 AI 生成功能，需要配置大模型 API Key。

## 八、项目特点

1. 面向真实教学场景，围绕教师教学和学生学习两个角色展开。
2. 采用桌面客户端与后端服务分离的方式，结构清晰，便于维护。
3. 引入大模型能力，提高题目生成和教学辅助效率。
4. 支持课程、小节、学习记录等教学数据管理。
5. 具备后续扩展为个性化学习分析平台的基础。
